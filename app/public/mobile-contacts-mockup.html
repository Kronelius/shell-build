Why this works

  • No horizontal scroll. Cards are width:100% with a fixed grid template; nothing inside can push the body wider.
  • Same data density. All eight columns of the desktop table are still represented — name, email, company-via-detail, lifecycle (badge), owner (avatar/ago line), tags, updated time. The columns we drop on the card view (Company name as a separate cell, Owner name) stay one tap away on the detail page.
  • Two buttons fit. Import CSV and + Add Contact are full-width flex:1 on mobile, side-by-side, instead of being shoved off the right edge.
  • Filters collapse. One pill with a count badge replaces six selects. The drawer reuses the existing FormField as="select" component — no new control work needed.
  • Bulk bar is reachable. Pinned to the bottom with safe-area padding, thumb-friendly. Doesn't push the list down when you toggle a checkbox.
  • Desktop is unchanged. The CSS toggles only inside @media (max-width: 640px). Same table renders ≥641px.
